# SocialPulse Web

Frontend React/Vite de SocialPulse.

## Lancement local

```sh
npm install
npm run dev
```

Le frontend attend l'API SocialPulse derriere `/api`, typiquement via Nginx ou Docker Compose.

## Build

```sh
npm run build
```

## Objectif

Cette application couvre la landing, l'authentification, les dashboards, les modules editoriaux et les integrations sociales de SocialPulse.
